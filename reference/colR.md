# Set default color palette

Set default color palette

## Usage

``` r
colR(cols = NULL, option = "", show = T, seed = 69)
```

## Arguments

- cols:

  Vector to set the default color palette. If "default", the default
  palette is reverted

- option:

  miscellaneous features. Includes "default" to reset to default
  palette, "shuffle" to shuffle the colors

## Value

Adds cancR_palette to the global environment

# Stepribbon geom for survival curves

Copy of the geom_stepribbon function by adibender/pammtools

## Usage

``` r
geom_stepribbon(
  mapping = NULL,
  data = NULL,
  stat = "identity",
  position = "identity",
  direction = "hv",
  na.rm = FALSE,
  show.legend = NA,
  inherit.aes = TRUE,
  ...
)
```

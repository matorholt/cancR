# population_denmark

Population tables for Denmark stratified on 5-year age groups and sex in
the period 1990-2025

## Usage

``` r
population_denmark
```

## Format

A data frame with 1296 rows and 4 columns:

- sex:

  sex, m=male, f=female

- age_group:

  age groups in 5-year intervals

- year:

  year

- population:

  number of inhabitants in Denmark in the given stratum
